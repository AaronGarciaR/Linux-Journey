## 📓 Bitácora de Comandos (Cheat Sheet)

### 📂 Navegación y Archivos
| Comando | Descripción (Inglés/Español) | Tip Pro |
| :--- | :--- | :--- |
| `pwd` | *Print Working Directory* / Ruta actual | Útil para saber dónde estás antes de borrar algo. |
| `ls -la` | *List all* / Listar todo | El `-a` muestra archivos ocultos (los que empiezan con punto). |
| `cd ~` | *Change Directory to Home* | Te lleva "a casa" (tu carpeta de usuario). |
| `mkdir` / `touch` | Crear carpetas / archivos | Puedes crear varios: `touch file{1..5}.txt` |

### 🔍 Búsqueda y Filtros
* **`grep` (Global Regular Expression Print):** Es como un buscador inteligente. 
  * *Ejemplo:* `cat archivo.txt | grep "error"` (Muestra solo las líneas que dicen "error").
* **`find`:** Para buscar archivos por nombre o tamaño.
* **`apropos`:** "Dime qué comando hace esto". Es el buscador de manuales.

### ⚡ Atajos de teclado
* `Tabulador`: Autocompletado (Tu mejor amigo para no escribir mal los nombres).
* `Ctrl + L`: Limpiar la terminal sin borrar el historial.
